# 项目架构

## 架构要点

- 编程语言
- 开发框架
- 开发规范：编码规范、项目结构和工作流
- 编译构建
- 最佳实践

    - 兼容性
    - 国际化
    - 屏幕适配
    - 错误处理：404、500
    - 性能优化
    - 埋点统计
    - 错误监控
    - 性能监控

- 自动化测试
- 持续集成和部署
