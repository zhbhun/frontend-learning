# 框架对比

## 小程序 vs vue

- 缺少类似 react hook 或 vue composition 这样的代码复用方式
- 缺少类似 react context 或 vue provide/inject 这样的组件通信方式
- 缺少状态管理框架
- slot 不支持 for 循环，且不支持传参

    - [微信小程序slot传参的坑及代替方案](https://www.jianshu.com/p/2041adde07be)

- 样式不支持预处理器
- 组件样式隔离规则过于复杂
- 代码文件太多，一个页面或组件需要定义四个文件
