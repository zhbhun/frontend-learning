### 业务状态 VS 应用状态 VS 组件状态

- 业务状态：类似于数据库表结构
- 应用状态：登录会话
- 组件状态：路由，表单，列表，表格

ps：每个页面独立维护对业务数据的索引，例如两个路由访问同一地址，各自维护私有的业务数据的引用。
