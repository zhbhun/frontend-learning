# 性能优化

- `shouldComponentUpdate`
- 使用压缩后的生产版本

    ps：React 默认包含了许多有用的警告信息。这些警告信息在开发过程中非常有帮助。然而这使得 React 变得更大且更慢，所以你需要确保部署时使用了生产版本。

- 使用开发者工具中的分析器对组件进行分析
- 虚拟化长列表
