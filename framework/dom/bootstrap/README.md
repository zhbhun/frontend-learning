1. 响应式设计：页面设计根据系统，屏幕尺寸，屏幕方向等进行相应的响应和调整；
2. 栅格系统：通过一系列的行与列的组合来创建页面布局；
3. 排版：优化标题，主体，文本，地址，列表，引用等；
4. 组件：无数可复用的组件，包括字体图标、下拉菜单、导航、警告框、弹出框等更多功能；
5. 插件：为 Bootstrap 的组件赋予了“生命”；
