  - 3 ×

    - 输入除号变为 3 ÷

  - 3 +
    - 输入乘号变为 3 ×
    - 输入除号变为 3 ÷
    - 输入加号变为 3 +
    - 输入减号变为 3 + -，在输入一个数字，变为 3 + -3 = 0
    - 输入减号变为 3 × -，在输入加号变为 3 +
  - 3 ÷：同 3 ×
  - 3 -：同 3 +
