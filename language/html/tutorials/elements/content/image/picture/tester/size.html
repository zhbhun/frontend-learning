固定高度

img=height: 100px

...
固定 img 高度 后，宽度会自适应

picture=height:100px

picture 默认是 inline 元素，无法设置大小，且 img 是作为一个子元素存在的

picture=display: inline-block; height: 100px

picture 及时设置为 inline-block，但固定高度时还是无法限制图片高度

picture=display: inline-block; height: 100px / img=width: 100%; height: 100%;

img 设置百分百宽高后，在部分浏览器生效，但是在 Safari 和 Android 一些旧浏览器上不生效（横向屏幕 100% 宽度）

picture=display: contents; / img=height: 100px;

将 picture 盒模型设置为 contents，然后样式都设置到 img 上，甚至支持响应式图片
