可以结合[Toolbox API](http://echarts.baidu.com/doc/doc.html#Toolbox)和实例学习echarts工具箱。
echarts工具箱自带的工具有：
- 辅助线标识
    - 辅助线开关
    - 删除辅助线
    - 清空辅助线
- 框选区域缩放
    - 区域缩放
    - 区域缩放后退
- 数据视图
- 动态类型切换
    - 直角坐标系
        - 折线图
        - 柱状图
        - 堆积
        - 平铺
- 还原
- 保存图片
- 自定义

注意点：一个图表最多只有一个工具箱，且一个工具箱必须有一个相关的图表
