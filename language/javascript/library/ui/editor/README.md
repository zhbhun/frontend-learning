# 编辑器

## 功能

- 工具栏

    - 文字

        - 字体
        - 字号
        - 段落
        - 对齐
        - 缩进
        - 行高
        - 加粗
        - 斜体
        - 方向
        - 上/下标
        - 下划线
        - 删除线
    
    - 颜色

        - 文字颜色
        - 背景颜色
        - 清除格式

    - 列表

        - 有序列表
        - 无序列表
        - 可选列表
    
    - 媒体

        - 图片
        - 视频
        - 文件

    - 画图

        - draw.io
        - CherryMindMap

    - 其他

        - 分割线
        - 引用
        - 表格
        - 公式
        - 表情
        - 链接
        - 代码
        - 撤销
        - 重做
