# Box Sample Asset

`Box.glb`、`Box.gltf` 和 `Box0.bin` 来自 Khronos Group 的 glTF Sample Assets：

- Source: https://github.com/KhronosGroup/glTF-Sample-Assets/tree/main/Models/Box
- Legal: © 2017, Cesium. CC BY 4.0 International.

`MultiNodeBox.gltf` 和 `BrokenExternalBuffer.gltf` 是本课程为了演示节点层级和错误路径制作的派生 glTF 文件，复用同一份 `Box0.bin` 几何数据。
