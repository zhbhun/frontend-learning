# Gaussian Splatting（3DGS）

- [SuperSplat](https://github.com/playcanvas/supersplat) - 3D Gaussian Splat Editor
- [SplatEdit](https://www.splatedit.app/) - SplatEdit.app is a browser-based editor for importing, editing, optimizing, and exporting 3D Gaussian Splats.
- [supersplat-viewer](https://github.com/playcanvas/supersplat-viewer) - User-friendly, high-performance viewer for 3D Gaussian splats
- [antimatter15/splat](https://github.com/antimatter15/splat) - WebGL 3D Gaussian Splat Viewer
- [Gaussian-Splatting-WebGL](https://github.com/kishimisu/Gaussian-Splatting-WebGL) - 3D Gaussian Splatting Renderer for WebGL
- [mkkellogg/GaussianSplats3D](https://github.com/mkkellogg/GaussianSplats3D) - Three.js-based implementation of 3D Gaussian splatting
- [Postshot](https://www.jawset.com/) - High Fidelity 3D Reconstruction on Windows
- [Scaniverse](https://dev.scaniverse.com/) - Scaniverse is a free 3D scanning app developed by Niantic that uses your smartphone's camera and sensors to create high-fidelity 3D models (meshes) and photorealistic 3D Gaussian splats. You can use it to capture objects, rooms, or entire outdoor environments and export them into formats like OBJ, FBX, or USDZ.
