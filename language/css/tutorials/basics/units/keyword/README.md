# 关键字

## inherit, initial, unset, revert

- initial：初始值，可以用于任意属性，表示样式属性值等于规范定义的默认值
- inherit：继承，只能用于可继承属性，表示样式属性值等于父元素的属性值
- unset：

    - 可继承属性：等同于 inherit
    - 不可继承属性：等同于 initial

- revert：类似 unset，但不可继承属性的值等于浏览器设定的值
