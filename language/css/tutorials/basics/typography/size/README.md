## 最小字体限制

Chrome 对特定的语言（汉语、日语、汉语等）限制了最小字体为 12px，早期可以通过 `text-size-adjust` 来解决，但现在已经废弃了，必须通过浏览器才能设置。

ps：Safari 和 Firefox 不存在该问题。
