## max/min-content、fit-content 和 fit-available

- fit-available：相等于 block
- fit-content：inline-block
- max-content：相等于 white-space 为 nowrap
- min-content：相当于 table-layout 为 auto 时，diaplay 为 table-cell 的单元格
