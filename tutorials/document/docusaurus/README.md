Docusaurus
========

- [官方网站](https://docusaurus.io/en/)
- [代码仓库](https://github.com/facebook/docusaurus)

## 安装使用

- `npm install --global docusaurus-init`
- `docusaurus-init`
- `mv docs-examples-from-docusaurus docs`
- `cd website`
- `mv blog-examples-from-docusaurus blog`
- `npm start`

## 网站结构

```
root-of-repo
├── docs
└── website
│   └── blog
│   └── core
│       └── Footer.js
│   └── node_modules
│   └── package.json
│   └── pages

```
