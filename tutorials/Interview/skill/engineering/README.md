## 代码审查

- 好处：通过代码审查，可以及早发现项目中存在的问题、促进同事之间的沟通与交流，并且可以在讨论中迸发出智慧的火花。
- 技巧

    - 使用特性分支
    - 将审查隔离为小的修改
    - 使用专门为代码审查而设计的工具
    - 检查清单
    - 有建设性的输入
    - 人人参与
    - 要有审查流程
